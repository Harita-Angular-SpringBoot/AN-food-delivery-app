# 🍽️ Food Delivery App

A full-stack food ordering application built using Angular and Spring Boot Microservices.
## 🛠 Tech Stack

- **Frontend**: Angular 19+
- **Backend**: Spring Boot (Java 17+)
- **Database**: MySQL
- **CI/CD**: Jenkins, Docker, SonarQube
- **Version Control**: Git & GitHub
## 📂 Project Structure

```bash

├── backend/
